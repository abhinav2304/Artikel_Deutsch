# Artikel Deutsch - German Vocabulary Learner

Welcome to the **Interactive German Vocabulary Learner**! This project is designed to help you master German vocabulary and internalize the tricky gender articles (der, die, das) through interactive tools and games.

## üöÄ How to Use the App

You don't need to install any special software or know how to code to use this app. It runs entirely in your web browser!

### Step 1: Open the App
1. Download or clone this repository to your computer.
2. Open the `german-learner` folder.
3. Double-click on the **`index.html`** file. 
4. It will automatically open in your default web browser (like Chrome, Safari, or Edge) and you can start learning immediately!

### Step 2: Explore the Features

* **Dashboard**: See your total word count and track your progress.
* **Article Game**: A fast-paced game where you have 60 seconds to match the correct article (Der, Die, or Das) to the German noun. It uses color-coding to help build visual memory.
* **Smart Flashcards**: Review words at your own pace. Click a card to flip it and reveal the translation, article, and plural form. You can filter the cards by category (e.g., "Lektion 11 Notes")!
* **Vocabulary List**: A complete, searchable dictionary of all 780+ words in the database. You can quickly look up any word in English or German.

## üìÅ What's in this Repository?

For those interested in how the app is built or where the data comes from:
* `german-learner/`: The folder containing the actual web application code (HTML, CSS, JS).
* `*.pdf`: The original German vocabulary lists.
* `*.txt`: The text extracted from the PDFs.
* `*.csv`: Spreadsheets containing the organized vocabulary data.
* `*.py`: Small Python scripts used to convert the PDF rules and handwritten notes into the interactive data used by the app.
